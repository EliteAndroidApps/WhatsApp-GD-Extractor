Heads up! To get an OAuth Token:
  1. Visit this page
  2. Click "Get OAuth Token"
  3. If the token is already there, clear it. We want to make sure you have the proper permissions
  4. Check "playlist-read-private", "playlist-read-collaborative", and "user-library-read"
  5. Click "Request Token"
  6. Either login, or if you're already logged in click "Okay"
  7. Copy the token in the "OAuth Token" field
